# 真实场景数据说明

## 1. 真实数据采集方式
本数据集由手机摄像头自行拍摄与淘宝或京东商品图片采集获取，未使用任何现有公开数据集。

## 2. 采集时间与采集场景
- 采集场景包括：室内（教室、实验室）、室外（校园道路）等不同光照和背景复杂度的场景。
- 包含了佩戴口罩、未佩戴口罩、以及由于角度/手部遮挡产生的不同样本。

## 3. 数据数量
- 共有 20 张以上图像，存放在 `images/` 文件夹下。

## 4. 是否包含人工标注
- 未提供人工标注，直接用于已训练模型的泛化测试。

## 5. 与原数据集的差异
- 背景更加复杂，包含特定的校园/室内光线干扰，部分目标处于远距离小尺寸状态。

## 6. 是否经过裁剪、缩放、增强等处理
- 仅保留了原始拍摄状态，未经过任何缩放或人工增强处理。

## 7. 数据来源声明
> 本组提交的真实场景数据由小组成员自行采集或制作，不包含任何课程所用公开数据集、网络公开标注数据集或已有实验数据集中的原始图像。
